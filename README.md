# Numerical Methods Suite — C#

A collection of numerical algorithms implemented from scratch in C#,
developed as part of a Numerical Methods course.

## What's implemented

### Interpolation
- **Lagrange polynomial interpolation** — approximates functions through a set of known points

### Numerical Integration
- **Rectangular method**
- **Trapezoid method**
- **Simpson's method**
- **Adaptive-grid Simpson** — adjusts grid size to meet precision constraints

### ODE Solvers
- **Euler's method**
- **Runge-Kutta 2nd order**
- **Runge-Kutta 3rd order**

## Notes
All methods implemented from scratch without numerical libraries.
Each module includes a written methodology report with results and error analysis.

## Language
C#
